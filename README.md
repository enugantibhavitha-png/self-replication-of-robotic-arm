# Self-Replicating Robotic Arm Using Microcontrollers

## Overview
Designed and developed a gesture-replicating robotic arm system 
that mimics human arm movements using microcontroller-based 
control logic and an Android application interface. The system 
enables a pick-and-place robotic arm with a soft capturing 
gripper to replicate directional movements — forward, backward, 
left, and right — transmitted wirelessly via an Android 
application. Applications include bomb disposal, hazardous 
environment operations, and autonomous manufacturing.

## Problem Statement
Existing robotic systems rely on remote control or manual 
programming for movement — they do not autonomously replicate 
human gestures in real time. This project addresses the need 
for an intuitive, gesture-driven robotic arm that can safely 
handle suspicious or hazardous objects without direct human 
contact, using wireless Android-based command delivery.

## System Architecture

### Hardware Components
| Component | Role |
|---|---|
| AtMega 168 Microcontroller | Primary processing and control unit |
| Servo Motors | Joint actuation and directional movement |
| Soft Capturing Gripper | Safe pick-and-place of delicate/suspicious objects |
| PLA Structural Frame | Lightweight, solvent-resistant arm body |
| Sensors | Real-time position and movement feedback |

### Software & Control
- **Android Application** — wireless command interface for 
  directional control
- **Embedded C** — low-level microcontroller programming
- **MATLAB** — simulation and control algorithm development
- **VS Code** — development environment
- **Sensor-Actuator Coordination** — real-time feedback loop

---

## Methodology

### Phase 1 — System Design
- Designed mechanical arm structure replicating human arm 
  joint behavior
- Defined pick-and-place task requirements and safety criteria
- Selected PLA material for structural frame — lightweight and 
  solvent-resistant

### Phase 2 — Control System Development
- Programmed AtMega 168 microcontroller with movement control 
  logic
- Developed Android application for wireless command transmission
- Implemented real-time sensor feedback for position accuracy

### Phase 3 — Prototype Build
- Assembled physical robotic arm with servo motor joints
- Integrated soft capturing gripper for safe object handling
- Connected microcontroller with Android command receiver

### Phase 4 — Testing & Validation
- Tested pick-and-place operations across multiple object types
- Validated wireless command accuracy and response time
- Conducted safety validation for hazardous object handling
- Documented experimental results and performance findings
