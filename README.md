# Self-Replication of Robotic Arm

This research project explores autonomous robotic replication concepts through the design and development of a self-replicating robotic arm prototype.

## Technologies Used
- MATLAB
- Embedded Systems
- Microcontrollers
- VS Code

## Key Contributions
- Developed control algorithms
- Implemented sensor-actuator coordination
- Built robotic arm prototype
- Conducted experimental validation
